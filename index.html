Instructions: You can search for text in any column using the search bar (except for descriptions), pressing Enter / Return will scroll you to the table. Click on a column header to sort by that column (both ascending and descending order are possible).

On smaller screens, some columns may be cut out. Click on the ship name to expand a row to view the hidden columns.

Alternatively, if you'd like to get the full table regardless of screen size, go to https://elegater.github.io?full=yes instead (it's a slightly different URL that you can bookmark to access the full size table by default)

You are currently viewing the version of the page with the table displaying all columns regardless of screen size (which will not entirely fit into smaller screens).

If you'd like to view the version that is more compatible for smaller screens, go to https://elegater.github.io/ instead (it's a slightly different URL that you can bookmark to access the mobile-compatible version by default)
